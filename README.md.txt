 Project Overview
This project predicts website session conversions using Google Analytics data, providing actionable insights for marketing and business optimization. It demonstrates an end-to-end machine learning workflow: from data cleaning and feature engineering to model training and deployment-ready serialization.

## Key Features
-Data Cleaning & Preprocessing:** Handle missing values, parse nested data, and transform features.
-Exploratory Data Analysis (EDA):** Visualize trends and key factors affecting user conversions using Pandas, Matplotlib, and Seaborn.
- Machine Learning Model:** Logistic Regression trained with Scikit-learn to predict user conversions accurately.
- Model Serialization:** Save trained model using Pickle for reuse without retraining.

Tech Stack
Python | Pandas | NumPy | Scikit-learn | Matplotlib | Seaborn | Pickle

 Workflow
1. Load and preprocess Google Analytics session data  
2. Perform EDA and visualize trends  
3. Train Logistic Regression model  
4. Serialize model using Pickle for reuse  

 Outcome
- Identified key user behaviors influencing conversions  
- Built a predictive model to support marketing and website strategy  
- Created a reproducible, production-ready ML workflow  

 How to Run
1. Clone the repository  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run the notebook or training script to reproduce results
