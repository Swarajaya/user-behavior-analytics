# user-behavior-analytics
Built an end-to-end ML project predicting website conversions using Google Analytics data. Performed data cleaning, feature engineering, and trained a Logistic Regression model, serialized with Pickle for reusable, production-ready predictions.

This project predicts user conversion using Google Analytics data with a Logistic Regression model.  

Try the live app here: https://huggingface.co/spaces/swarajaya/user-behavior-analytics
